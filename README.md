# Welcome to Web Development! 🌐

## What is This?

This is your personal workspace for learning HTML and CSS—the building blocks of every website you've ever visited! You'll build real web pages while learning to work with AI tools like GitHub Copilot.

## Getting Started

### First Time Setup

1. **Make sure you have:**
   - VS Code installed ✅
   - GitHub Copilot enabled ✅
   - Live Server extension installed ✅

2. **Read the Student Guide:**
   - Open [STUDENT_GUIDE.md](STUDENT_GUIDE.md) and read it!
   - It explains how to work with your AI tutor
   - Shows you how to ask good questions

3. **Start with Lesson 1:**
   - Go to `lessons/01-html-basics/`
   - Read the README.md
   - Open exercise.html
   - Start coding!

## How This Works

**You have an AI tutor!** GitHub Copilot in this workspace is configured to help you learn, not just give you answers. It will:
- Explain concepts clearly
- Give you hints when you're stuck
- Ask questions to help you think
- Check your work and provide feedback

**Important:** The AI won't just do your work for you. That's on purpose! You learn by figuring things out, not by copying answers.

## Quick Tips

**When you need help:**
- Read the lesson README first
- Try something before asking
- Ask specific questions: "How do I center text?" not "Help"
- Use the magic commands: "Check my work", "Give me a challenge", "I'm stuck"

**While you code:**
- Use Live Server to see changes instantly (right-click HTML file → Open with Live Server)
- Save often (Cmd+S / Ctrl+S)
- Commit your progress with git regularly
- Experiment! Change things to see what happens

**Remember:**
- ✅ Struggling is normal (and good!)
- ✅ Mistakes are how you learn
- ✅ Ask questions when stuck
- ✅ Take breaks when frustrated
- ❌ Don't just ask for complete answers
- ❌ Don't copy without understanding
