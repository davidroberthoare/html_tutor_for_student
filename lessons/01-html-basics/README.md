# Lesson 1: HTML Basics

## Learning Objectives

In this lesson, you will learn:
- The structure of an HTML document
- How to use basic HTML tags
- The difference between the head and body sections
- How to create a valid HTML5 document

## Instructions

1. Open `exercise.html` in VS Code
2. You'll see a partially completed HTML document with some blanks marked by comments
3. Fill in the missing parts to create a complete, valid HTML document
4. Save your work and open the file in a browser to see the result
5. Use Live Server extension for instant preview (right-click → Open with Live Server)

## What You're Building

A simple "About Me" page with:
- Proper HTML5 structure
- A title that appears in the browser tab
- A main heading with your name
- A paragraph about yourself

## Success Criteria

Your page should:
- [ ] Have a DOCTYPE declaration at the top
- [ ] Have html, head, and body tags properly nested
- [ ] Include a title in the head section
- [ ] Show a heading and paragraph in the browser
- [ ] Have all tags properly closed
- [ ] Be properly indented

2. You should see your name as a heading and your paragraph
3. Check the browser tab—it should show your title
4. If something doesn't appear, check for missing or incorrect tags
