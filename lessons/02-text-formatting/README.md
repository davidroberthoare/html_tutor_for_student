# Lesson 2: Text Formatting

## Learning Objectives

In this lesson, you will learn:
- How to use heading levels (h1 through h6)
- How to create paragraphs
- How to make ordered and unordered lists
- How to emphasize text with strong and em tags
- When to use different types of text formatting

## Instructions

1. Open `exercise.html` in VS Code
2. Complete the partially-written HTML document
3. Add the required elements where indicated by TODO comments

5. Preview your work using Live Server

## What You're Building

An "About Me" page with:
- A clear heading hierarchy
- Multiple paragraphs about yourself
- A list of your hobbies or interests
- Emphasized text using strong and em tags

## Success Criteria

Your page should:
- [ ] Use h1 for the main title
- [ ] Use h2 for section headings
- [ ] Have at least 2 paragraphs
- [ ] Include one unordered list with at least 3 items
- [ ] Use strong or em tags to emphasize important words
- [ ] Have proper nesting (list items inside list containers)
- [ ] Be well-formatted and easy to read

## Key Concepts to Remember

- `<h1>` is for your main page title (use only one per page)
- `<h2>` through `<h6>` are for subheadings (don't skip levels!)
- `<ul>` creates a bulleted list, `<ol>` creates a numbered list
- `<li>` are the items that go inside lists
- `<strong>` makes text important (bold by default)
- `<em>` emphasizes text (italic by default)

## Common Mistakes to Avoid

- Don't put `<li>` tags outside of `<ul>` or `<ol>`
- Don't skip heading levels (don't go from h1 to h4)
- Remember: lists need THREE closing tags minimum (each li, then the ul/ol)
