# Personal Portfolio

## Project Overview

Build a personal portfolio website to showcase yourself, your skills, and your projects.

## Suggested Sections

1. **Hero/Intro** - Eye-catching intro with your name and tagline
2. **About Me** - Who you are, what you do, what you're interested in
3. **Skills** - List of your skills (HTML, CSS, etc.)
4. **Projects** - Showcase projects you've built (even these lessons!)
5. **Contact** - How to reach you (could be social media links)

## Features to Include

- Navigation bar
- Header with your name
- Multiple sections with clear headings
- Images (your photo, project screenshots)
- Links to your GitHub, social media, etc.
- Styled cards or sections
- Footer with copyright info

## Design Tips

- Keep it professional but show personality
- Use a consistent color scheme
- Make sure text is readable
- Include visual hierarchy (different heading sizes)
- Don't overcrowd—white space is good!

## Getting Started

1. Plan your sections on paper
2. Gather content (write your bio, collect images)
3. Choose colors and fonts
4. Build HTML structure first
5. Style with CSS
6. Polish and refine

## Stretch Goals

- Add a contact form (HTML only, won't send yet)
- Make it responsive
- Add smooth scrolling to sections
- Include project cards with hover effects
- Add your resume as a downloadable PDF link
