# Simple Blog

## Project Overview

Create a blog-style website with multiple posts or articles. Write about topics you care about!

## Suggested Structure

1. **Homepage** - List of recent posts
2. **Individual Posts** - Full article sections
3. **About** - About the blog/author
4. **Sidebar** (optional) - Categories, recent posts
5. **Navigation** - Easy access to sections

## Blog Post Elements

Each post should have:
- Title (h2)
- Date published
- Author (you!)
- Featured image (optional)
- Body text with paragraphs
- Maybe tags or categories

## Design Tips

- Easy to read (good typography!)
- Clear hierarchy (title > subtitle > body)
- Comfortable line length (not too wide)
- Good line spacing
- Visual breaks between posts

## Blog Topics Ideas

- Learning journey (documenting what you're learning)
- Game reviews or strategies
- Book reviews
- Movie/TV show thoughts
- Hobby tutorials
- Tech tips
- Creative writing
- Opinion pieces
- How-to guides

## Getting Started

1. Write 3-5 blog posts (200-300 words each)
2. Find or create header images
3. Plan layout (single column? sidebar?)
4. Build post structure
5. Style for readability

## Layout Options

- Single column (simple, clean)
- Main content + sidebar
- Card-based layout
- Magazine-style with featured post

## Stretch Goals

- Add a search bar (visual only)
- Category pages
- "Related posts" section
- Comment section (visual placeholder)
- Social share buttons
- RSS feed icon
